Instructions for using the Course websites

Welcome to the Canvas website for PHWM2998 - Introduction to R Programming for Epidemiologic Research. To complete this course, you will actually need to use a few additional websites as well. This may cause a little bit of confusion at first, but my hope is that it will allow you to access the best available tools for learning R programming and it will allow you to continue to access course resources long after the semester is completed.

Here is a list of the websites we will use in this course along with a brief overview of how we will use them:

  • Canvas: Submit assignments and quizzes, and view your grades on assignments and quizzes
  • Introduction to R Programming for Epidemiologic Research Course Website: View the course outline, view individual learning modules, and access links to course resources
  • Slack: Our primary communication tool for this class
  • DataCamp: A tool for practicing your R programming online and getting real-time feedback

You should not have to do anything special to access the course website -- just click on the link above. I also recommend that you bookmark the course website in your browser of choice.

Before you can use Slack or DataCamp, you will need to sign up for a free account with both. Below are some instructions for doing so.

Using Slack

What is Slack?

Slack is a tool for communication, collaboration, and sharing resources. For the purposes of our class, you can think of Slack as a replacement for email, discussion boards, and cloud-based file sharing (e.g., Dropbox). If you are interested, you can read a short introduction to Slack article here, or view a short demo video here.

Why are we using Slack?

"Okay, Brad. Slack looks cool, but why not just use the email, discussion boards, and cloud-based file sharing capabilities that Canvas already has?" The primary reason is because it benefits YOU! At the end of the semester, you will lose access to course-related materials that are on Canvas. However, you will be able to access Slack for the foreseeable future.

How do I access the course workspace on Slack?

I've already created a Slack workspace for this course, which is accessible at https://intro-r-prog.slack.com. You can read about joining a Slack workspace here. Keep in mind, you must use your "@uth.tmc.edu" email address to sign up for our Slack workspace.

Additional questions? Go ahead and log into Slack and post them in the #general channel!

Using DataCamp

What is DataCamp?

DataCamp is an incredible resource for interactively learning to program in R, Python, and SQL. The website is broken into topic-specific courses that include videos, readings, and interactive coding sessions that provide you with immediate feedback about any errors in your code. Typically, you have to pay a monthly or yearly subscription fee to access DataCamp courses. However, DataCamp for the Classroom allows me to provide you with access to all of DataCamp's courses free of charge for 6 months. All you have to do is register!
