Welcome to Intro to R Programming for Epidemiological Research!

This first module is about orienting you to the course and making sure we all have a common vocabulary to speak to each other with. At this point, we aren’t assuming that you know anything about the R programming language or even programming in general. At this point, all the programming you do will be on the DataCamp.com website in a virtual programming environment (a computer in “the cloud”).

This module will cover the following topics:

  • Using the Canvas course
  • Student introductions
  • Creating prerequisite online accounts
  • How to use DataCamp
  • Introduction to the R programming language
  • Installing R and RStudio

By the end of this module you should be able to:

  1. Describe what this course is about
  2. Locate important information on the course syllabus
  3. Create and submit simple R programs on the DataCamp learning platform
  4. Install and open R and RStudio on your personal computer
